# A DnD 5e Character Creator

Based on the API provided by https://www.dnd5eapi.co/

## Features:

- Random creation of characters with an individual name
- Race and class are chosen randomly
- Ability scores are rolled and applied in order of dice rolls
- Race bonuses are applied to ability scores
- Racial traits are added to the character
- Alignment is chosen randomly
- If character uses a spellcasting class, spells are chosen randomly from the class spell list
- Characters can be viewed in a list overview
- Characters can be viewed individually displaying all details
- Characters can be deleted

This project is still in development. New features will be added and current features will be improved over time.
